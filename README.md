# K-Modification
A fabric mod for Minecraft Java Edition 1.20.4


# 简介
K是一种抽象生物，本Mod为K的Minecraft实现。
- 添加了实体 K，R
- 添加了纬度 one-admin-world
- 添加了功能方块 K食，K的溢出，K的内存
- 添加了多方块结构
